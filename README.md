# FinStart test site

GitHub Pages static preview:

- Site: `https://th0omassed-bot.github.io/finstart-test/`
- Test admin: `https://th0omassed-bot.github.io/finstart-test/admin-finstart-panel/`

The GitHub Pages admin is a browser-only test mode. It saves changes to `localStorage` in the same browser so the site can be checked without PHP hosting.

For a real production admin panel, deploy the project to PHP hosting and create `admin-finstart-panel/config.php` from `admin-finstart-panel/config.example.php`.
